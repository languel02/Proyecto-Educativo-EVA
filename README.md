# Proyecto-Educativo-EVA
Plataforma educativa Web orientada a mantener interacciones solidas entre profesores y estudiantes a través de asignaciones, contenidos, cursos, evaluaciones y mas.

La app esta adaptada para utilizarse a nivel web desktop y movil con una interfaz amigable y de facil manejo para los usuarios.

La misma fue desarrolla utilizando HTML, CSS, y JAVASCRIPT.
